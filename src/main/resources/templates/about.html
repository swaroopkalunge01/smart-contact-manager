About Smart Contact Manager

Smart Contact Manager helps you store, manage & organize your contacts securely.

🔥 Key Features

  • ✔ Store unlimited contacts
  • ✔ Upload contact images
  • ✔ Spring Security based authentication
  • ✔ Fast search & filter
  • ✔ Role-based access

🛠 Technologies Used

  • Spring Boot
  • Spring MVC + Thymeleaf
  • Spring Security
  • Hibernate + JPA
  • MySQL
  • Bootstrap 4
